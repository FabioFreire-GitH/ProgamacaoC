# 📑 Estudos de Lógica de Programação e Estrutura de Dados em C

Este repositório foi criado em 2025 para documentar a minha base de lógica de programação e os meus primeiros passos na sintaxe da linguagem C, servindo como alicerce fundamental para a minha transição de carreira em tecnologia.

## 🚀 O que foi explorado até o momento:
- **Lógica e Aritmética:** Criação de algoritmos matemáticos e manipulação de variáveis.
- **Estruturas de Controle:** Domínio de condicionais (`if/else`) e laços de repetição (`while`, `for`).
- **Modularização:** Construção de funções e procedimentos para reaproveitamento de código.
- **Estruturas de Dados:** Implementação e manipulação de `structs` para organização de dados complexos.

## 🛠️ Como executar os códigos:
Para rodar qualquer um dos programas, é necessário ter um compilador C (como o GCC) instalado.
